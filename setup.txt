| `androidstorage4kivy` | Read/write files in Android storage |
| `plyer` | Access GPS, SMS, Camera, and other device features |

| `matplotlib` | Creating charts in Kivy |
| `numpy` | Required by `matplotlib` for efficient calculations |
| `pyjnius` | Interact with Android Java APIs (SMS, Camera, etc.) |
| `pyproj` | Geospatial calculations (for maps) |
| `geopy` | Geolocation support for GPS |

## **üìÑ 2Ô∏è‚É£ Updated `android.permissions`**
```ini
android.permissions = INTERNET, ACCESS_FINE_LOCATION, ACCESS_COARSE_LOCATION, READ_EXTERNAL_STORAGE, WRITE_EXTERNAL_STORAGE, READ_MEDIA_IMAGES, READ_MEDIA_VIDEO, READ_MEDIA_AUDIO, CAMERA, SEND_SMS, RECEIVE_SMS
```

### **‚úÖ Explanation of Each Permission**
| **Permission** | **Purpose** |
|------------|-------------|
| `INTERNET` | Required for fetching data (e.g., API requests) |
| `ACCESS_FINE_LOCATION` | Get precise GPS location |
| `ACCESS_COARSE_LOCATION` | Get approximate location |
| `READ_EXTERNAL_STORAGE` | Read files from storage |
| `WRITE_EXTERNAL_STORAGE` | Save files to storage |
| `READ_MEDIA_IMAGES` | Read images from gallery (Android 13+) |
| `READ_MEDIA_VIDEO` | Read videos from gallery (Android 13+) |
| `READ_MEDIA_AUDIO` | Read audio files (Android 13+) |
| `CAMERA` | Take a picture using the device camera |
| `SEND_SMS` | Send messages from the app |
| `RECEIVE_SMS` | Detect incoming SMS |

## **üìÑ 3Ô∏è‚É£ Additional Configurations for Buildozer**
You must enable **Android API level 33+** for newer permissions:
```ini
android.api = 33
android.minapi = 21
android.ndk = 23b
android.arch = arm64-v8a
```

---

## **üìÑ 4Ô∏è‚É£ Enable SMS and Camera Features**
For **SMS and Camera**, add these to your `buildozer.spec`:
```ini
android.features = android.hardware.camera, android.hardware.camera.autofocus
android.allow_backup = 1
android.backup_rules = True
```

---

## **üìÑ 5Ô∏è‚É£ Enable Maps (KivyMapView)**
If you're using **Kivy MapView**, add:
```ini
requirements = python3, kivy, git+https://github.com/kivymd/KivyMD.git, requests, androidstorage4kivy, plyer, pillow, matplotlib, numpy, pyjnius, pyproj, geopy, kivy-garden.mapview
```
Then install it in your project:
```sh
garden install mapview
```
